🎯 Interaction Patterns & Micro-animations

Enhanced User Feedback

🔄 State Transitions

  • Card hover: 300ms ease-out transform and shadow
  • Button press: 150ms scale(0.98) feedback
  • Tab switching: 300ms slide transition
  • Token selection: 200ms color and elevation change

📲 Touch Feedback

  • Ripple effect on button press
  • Haptic feedback for mobile interactions
  • Visual press state with opacity change
  • Smooth scroll behavior for long content

⚡ Loading Animations

  • Skeleton shimmer: 1.5s infinite loop
  • Progress bars: Smooth width transitions
  • Spinner rotation: 1s linear infinite
  • Content fade-in: 300ms opacity transition
