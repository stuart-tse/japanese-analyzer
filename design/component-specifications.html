Section Content
Header Word, readings, audio controls

Progressive Disclosure Strategy:
  • Always show: Translation, POS, basic info
  • Beginner mode: Expand Level 1 by default
  • Intermediate mode: Show Levels 1-2
  • Advanced mode: All levels available
  • Remember user's expansion preferences
interface DetailPanelProps { wordDetail: WordDetail | null; learningMode: 'beginner' | 'intermediate' | 'advanced'; onTTS: (text: string, language: 'ja' | 'zh') => void; onBookmark: (word: string) => void; isLoading: boolean; expandedSections: string[]; onSectionToggle: (section: string) => void; } interface WordDetail { originalWord: string; chineseTranslation: string; pos: string; furigana?: string; romaji?: string; jlptLevel?: string; frequency?: string; explanation: string; examples: Example[]; culturalContext?: string; }

Pattern: [Topic]は [Adjective][Noun]です[Particle]

Component Specifications

Feature Description
Multi-language English, Chinese, Grammar analysis
Contextual hints Cultural and usage context
Confidence scores Show translation reliability
Alternative translations Multiple interpretation options
Copy functionality Easy text copying for study
Translation Strategy:
  • Prioritize natural, contextual translations
  • Show literal vs. natural interpretations
  • Include cultural context when relevant
  • Highlight difficult-to-translate concepts
  • Provide alternative meanings for ambiguous text
